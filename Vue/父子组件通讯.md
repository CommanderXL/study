## 父子组件通讯

还是从一个实例出发：

```javascript
// 模板
<div id="app">
  <child-component :message="val"></child-component>
</div>


// js

Vue.component('child-component', {
  props: ['message']
  template: '<div>this is child component, I have {{message}}</div>',
})

new Vue({
  el: '#app',
  data() {
    return {
      val: 'parent val'
    }
  }
})
```

最终页面渲染出的内容为:

```javascript
this is child component, I have parent val
```
